## Notes for Porting

- This list represents the complete set of grammar rules that need to be ported
- Each rule may have multiple production alternatives
- Rules reference each other creating a complex dependency graph
- Terminal symbols (keywords, operators) are in UPPERCASE in the actual grammar
- Non-terminals (the rules listed above) are in lowercase or CamelCase
